sa # Dalliat App

Aplicación Flutter simple con pestañas de Login y Registro.

- Login: usuario y contraseña.
- Registro: usuario, contraseña, correo y fecha de nacimiento.
- Selector de fecha para fecha de nacimiento.
- Diseño básico con pestañas.

## Uso

Ejecutar con `flutter run` en dispositivo o emulador.

## Estructura

- `lib/main.dart`: código principal.
