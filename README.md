# Dalliat App

Aplicación Flutter de Login y Registro.

- Login: usuario y contraseña.
- Registro: usuario, contraseña, correo y fecha de nacimiento.
- Diseño básico con 2 pestañas dezlizables y contraseña oculta.


## Estructura

- `lib/main.dart`: código principal.

##Funcion
--En la pestaña LogIn se te pediran los datos que previamente el usuario puso en el register especificamente:
Usuario
Contraseña

--En la pestaña Register se pediran los siguientes datos para su validacion
Usuario
Contraseña
Correo
Fecha de nacimiento

--Estos menus incluyen:
Deslizable entre las opciones register y Login
Tapar las claves

--uso de almenos 7 ramas
